# Muhammad Abubakar · Portfolio  

A clean, responsive portfolio website for **Muhammad Abubakar** – a Dispenser Technician and Pharmacy Technician.  
The page showcases his professional profile, 4 educational certificates, core skills, and includes an interactive 3D medical cross model built with Three.js.
