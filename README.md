# Muhammad Abubakar · Portfolio

A fully animated portfolio for **Muhammad Abubakar** – Dispenser Technician & Pharmacy Technician.

🔗 **Live Demo:** [https://m-abubakar-portfolio.vercel.app](https://m-abubakar-portfolio.vercel.app)

## ✨ Features

- **Typing Effect** on hero subtitle
- **Floating Medical Icons** in background
- **Mouse Parallax** on 3D cross
- **Animated Counters** (4+ certifications)
- **Scroll Reveal** with stagger effects
- **Hover animations** on cards, buttons, and skill items
- **Fully Responsive** for all devices

## 🚀 Getting Started

1. Clone the repo and add your images to `images/` folder.
2. Open `index.html` in your browser.
3. Customize contact details and text as needed.

## 📦 Deploy

Push to GitHub and import to Vercel as a static site.
