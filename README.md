## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/m-abubakar-portfolio.git
cd m-abubakar-portfolio
2. Add Your Images
Place your images in the images/ folder with these exact names:

profile.jpg – Profile picture (square, 300×300 recommended)

cert1.jpg – Certificate 1 (Community Action For Disaster Response)

cert2.jpg – Certificate 2 (Basic Community Safety Online Training)

cert3.jpg – Certificate 3 (Certificate of Achievement)

cert4.jpg – Certificate 4 (Pharmacy Technician Diploma)

3. Open Locally
Simply open index.html in your browser – no build step required.

4. Customize Content
Profile text: Edit the .about-text p section in index.html

Contact details: Update email, phone, and social links in the #contact section

Colors: Edit CSS variables inside :root in style.css

📦 Deploy to Vercel
Push the code to a GitHub repository.

Go to Vercel and import the project.

Select the repository and keep default settings (static site).

Deploy – your portfolio will be live in seconds.

## ✅ How to Use

1. **Create folders:** `css/`, `js/`, `images/`
2. **Copy code:**
   - `index.html` → paste in root
   - `style.css` → paste in `css/` folder
   - `script.js` → paste in `js/` folder
   - `README.md` → paste in root
3. **Add images:**
   - `profile.jpg` → `images/` folder
   - `cert1.jpg` to `cert4.jpg` → `images/` folder
4. **Open `index.html`** in browser – images will show automatically.
